# math API
由xiwangly编写，用于访问该接口快速运算一些值，请保留著作权，还在不断改进中……
教程（必看）：  
<https://www.lolichan.vip/threads/qrspeed-6.53/>或<https://www.lolichan.vip/threads/api-xiwangly-api.205/>  
本例常用url get 规则：  
<http://域名/math.php?x=x的值&y=y的值&z=z的值&m=计算方式>  

PHP版本需\>7.0
